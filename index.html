Loops

While Loop

This loop is gonna run as long as a condition is true. Most of the 'while' loops we're gonna see in this unit is gonna be with 'counters' (or contadores in Spanish).

Counters mean that we will increase a variable by +1 every time we run through the loop.

Example: i = 0
while i < 5:
    print("hi")
    i += 1

In the example, we increase 'i' by +1 every time we run through the loop until we reach 5; meaning our while loop will stop once 'i' becomes greater than 5.

Extra:

I haven't seen the teacher use this, but if he asks, you'll already know. There's something we call 'while true loops' or just infinite loops.

Example:
while True:
    print("hi")

In this case, we're looping while 'True'. Since True isn't a variable that we're ever gonna change, it will always stay True, meaning the loop will never stop.

For Loop

For loops iterate over a sequence. Oh shit, I forgot you're dumb asf, I'll make this sound easier for you.

For loops are while loops but without the need of us having to increase a variable to keep track if it's higher than what's needed. Still not getting it?

Maybe this is simpler: In for loops, you can set a specific amount of times a loop will repeat without the need of a variable that will increase every time we run through the loop. Stiiiiiiiiill not getting it? Well sucks, I don't find any other way to explain it rather than with examples:

Example:
for i in range(5):
    print(i)

In this example, 'i' is our temporary variable. Now, some questions you might have: What do u mean temporary? What's the difference between the 'i' in the while loop and this one?

I'll answer both of 'em rn: I say temporary because this variable is only used for the loop, we're not setting it BEFORE the loop. Now the difference is that 'i' in the for loop is only there to keep track of the times we've repeated the loop. In the while loop, we normally have to increase it by +1 or check if the condition is True before running the loop again.

Example 2:
repetitions = int(input("Enter a number: "))
for i in range(repetitions):
    print(i)

In this case, we grab the user's input as an int (whole number), and we store it in the 'repetitions' variable. Then in the for loop, we use the 'repetitions' variable as our range (the times we will repeat it basically) and print 'i' the amount of repetitions the user input.

Now, here's another thing you might notice in the for loop: Two numbers in the 'range' thing, like this: for i in range(0, 6). This is very easy to understand though. The first number specifies where the loop will start, and the second number where the loop will end. Not getting it again? Figured bish. Here's an example:

Example 3:
for i in range(3, 6):
    print(i)

In this case, this would be our output:

3
4
5

"Oh, but why does it print until 5 and not 6???????" I got the ACTUAL explanation for you. I wish the teacher explained this, but he's too stupid to explain something so simple.

The way Python indexes for loops is quite awkward, but it's better for actual people that program Python professionally. Here's the explanation anyways:

This is the range function with the two parameters we know: range(start, stop).

Start is the number where the for loop will start from, and stop is the number it has to reach to stop looping. The function works in this way: The start value is included (starts from here), and the stop value is excluded (goes up to, but does not include this). This is what's happening with Example 3: for i in range(3, 6) prints 3, 4, and 5. It stops just before 6.
